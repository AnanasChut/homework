## Работа со списками

Для создания списков можно использовать перед элементом звездочку(*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Для создания нуммерованных списком досточно исползовать цифры. Например, вот так:
1. Пункт 1
2. Пункт 2
